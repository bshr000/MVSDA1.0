# 可见光-SAR遥感多模态机场检测数据集（MVSDA 1.0）

## 1. 数据集简介
本数据集用于遥感多模态目标检测任务，包含可见光（RGB）与 SAR 两种模态数据，并提供 VOC旋转框 与 YOLO水平框 两种标注格式。

## 3. 下载地址
- 数据下载：https://www.scidb.cn/detail?dataSetId=29765a93b1c54e3d9fac27dbc03b8214&version=V1

## 4. 数据划分
- 比例：`train:val:test = 8:1:1`
- 划分文件：`scripts/train.txt`、`scripts/val.txt`、`scripts/test.txt`
- 划分文件格式：每行为文件名（含扩展名）
