# 可见光-SAR遥感多模态机场检测数据集（MVSDA 1.0）

## 1. 数据集简介
本数据集用于遥感机场目标检测任务，包含可见光（RGB）与 SAR 两种模态数据，并提供 VOC 与 YOLO 两种标注格式。
